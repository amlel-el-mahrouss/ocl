# OCL.Core

Core set of containers from the OCL.

## Requirements:

The OCL requires the following:

- [Boost](https://www.boost.org/)
- [Clang](https://clang.llvm.org/)
- [CMake](https://cmake.org/)
- [Git](https://git-scm.com/)
